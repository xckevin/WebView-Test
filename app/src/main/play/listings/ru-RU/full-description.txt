WebViewTest — практичный инструмент тестирования Android WebView для разработчиков, QA-инженеров и команд, работающих с гибридными приложениями или мобильными веб-страницами.

С его помощью можно загрузить URL, отсканировать QR-код, открыть локальный HTML-файл, изменить поведение WebView и проверить страницу прямо на реальном устройстве.

Основные возможности:

- Загрузка http и https страниц с нормализацией URL.
- Сканирование QR-кодов для быстрого открытия тестовых страниц.
- Открытие локальных HTML-файлов через системный выбор файлов.
- Сохранение недавней истории удаленных URL и локальных файлов.
- Переключение JavaScript, DOM Storage, cookies, сторонних cookies, кеша, mixed content, desktop mode, User-Agent, цветового режима и полноэкранного режима.
- Тестирование разрешений веб-камеры, микрофона, выбора файлов и геолокации.
- Просмотр console logs, ошибок страницы, cookies, storage, HTML source, упрощенных элементов, результатов JavaScript, запросов и загрузок.
- Ручное включение WebView debugging для release builds в Settings с последующей проверкой через desktop Chrome.

WebViewTest предназначен для тестирования и отладки. Это не универсальный веб-браузер, и он не связан с Google, Android или сайтом Android Developers.
