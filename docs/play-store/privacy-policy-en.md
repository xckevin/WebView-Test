# Privacy Policy for WebViewTest

Effective date: 2026-06-09

WebViewTest is an Android WebView testing and debugging tool. It is designed for developers and testers who need to load, inspect, and troubleshoot web pages inside Android WebView.

## Data we collect

WebViewTest does not require an account and does not intentionally collect personal data on a developer-operated server.

The app stores some information locally on your device:

- Recent test history, such as URLs and local file references.
- WebView settings selected by you.
- WebView data created by pages you load, such as cookies, cache, localStorage, and sessionStorage.

## Pages you load

You can load arbitrary websites in WebViewTest. Those websites may collect, process, or store data according to their own privacy policies. WebViewTest does not control third-party websites.

## Permissions

WebViewTest may request:

- Camera access for QR code scanning and for web pages that request camera capture.
- Microphone access for web pages that request audio capture.
- Location access for web pages that request geolocation.
- Internet access to load remote pages.

Camera, microphone, and location access require Android permission approval and are used only for the feature or web page request that triggered them.

## Data sharing

WebViewTest does not intentionally share your data with a developer-operated server. Data may be sent to websites that you choose to load, because those pages run inside Android WebView.

## Deletion

You can clear history, cookies, cache, and WebView storage from the app's settings/debug tools or from Android system settings.
